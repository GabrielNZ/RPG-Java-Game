# 👑 RPG Java Game

### Sobre o projeto

Algum tempo atras como hobby, decidi criar um jogo RPG no proprio console da IDE Java, com as mecanicas mais marcantes que todo RPG e D&D possuem.
Com apenas Java e POO, de forma discontraida construi um jogo funcional e com difersas mecanicas desde *Level* ate *Lutas* e *Loja*.

### ⚙️ Mecanicas 

- 🎲 Combate em Turno
- 👥 Diversas Classes
- 🧙‍♂️ Magias & Habilidades de Classe
- 🏰 Dungeons
- 🎁 Loots
- 🪙 Economia & Loja
- ✨ Level up
- 🛡️ Equipamentos & Armaduras
- 🧪 Itens Especiais
- 💼 Inventario Dinamico (Equipar Itens)

### ⚔️ Combate
Todo dano só será causado caso o ataque acerte. Seu resultado de dano diminui pela armadura do alvo (`Dano - Resistencia`)
Crítico caso o valor do dado seja `20`. 
Magias e habilidades podem ter efeitos que:
- Pulam rounds
- Neutralizam a armadura do oponente
- Roubam equipamentos e armamentos do openente para o uso proprio
- Curas e regeneração de mana
- etc...

# 🚦 Como Jogar o Jogo/Projeto
### 🛠 Pré-requisitos
- Git
- IDE Java
### 🚀 Como jogar na minha máquina?
1. Clone o repositório: `git clone https://github.com/GabrielNZ/RPG-Java-Game.git` && `cd RPG-Java-Game`.
2. Selecione a classe `Jogar` e inicie.
